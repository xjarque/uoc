Obtener los datos

Usaremos dos archivos para el anlisis. 1. El fichero medications, que tiene los diagnsticos donde seleccionaremos los pacientes con diabetes tipo 2. 2. El fichero labs, que tiene todas las medidas de laboratorio que sern nuestros predictores.

medications <- fread('medications.csv')
labs <- fread('labs.csv')

Definimos la variable dependiente

El cdigo ICD-10 para la diabetis melitus Tipus 2 es el E11, de forma que crearemos la variable is_diabetic para definir aquellos que tienen este cdigo.

Los cdigos de diagnstico ICD-10 se pueden encontrar en el ICD-10 Browser of the World Health Organization

diabetes <- medications %>%
  mutate(
    is_diabetic = ifelse(RXDRSC1 == 'E11', 1, 0)
  ) %>%
  select(
    SEQN,
    is_diabetic
  ) %>%
  unique() %>%
  left_join(labs)
## Joining, by = "SEQN"
outcome <- c('is_diabetic')

Definimos las variables independientes o predictores

Todas las variables de laboratorio que tenemos disponibles lo sern.

predictors <- names(diabetes)[!names(diabetes) %in% outcome]

Construimos el modelo

Entrenaremos un modelo XGBoost.

mx.diabetes <- as.matrix(diabetes, rownames.force = NA)
train_dmatrix <- xgb.DMatrix(data = mx.diabetes[, predictors], label = mx.diabetes[, outcome])

fit.xgb <- xgboost(
  data = train_dmatrix,
  nrounds = 10,
  objective = "binary:logistic"
)
